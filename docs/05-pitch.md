# Pitch
No vídeo eu explico sobre meu projeto de organizar minha vida financeira. Primeiro, o foco é quitar a dívida do cartão de crédito, entendendo quanto estou devendo e criando um plano mês a mês para conseguir pagar tudo.

Depois de zerar essa dívida, o próximo passo é começar a guardar dinheiro, com a meta inicial de juntar R$ 1.000,00. A ideia é criar disciplina financeira, cortar gastos desnecessários e ter mais controle sobre o dinheiro.

O projeto mostra uma mudança de mentalidade: sair das dívidas, se organizar e começar a construir uma reserva.
